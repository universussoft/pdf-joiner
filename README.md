# PDF Joiner

A free, open-source PDF merger that works entirely in your browser. No uploads, no server processing, 100% client-side.

## ✨ Features

- 🔒 **Privacy-focused**: All processing happens locally in your browser
- 🚀 **No uploads**: Files never leave your computer
- 📄 **Merge multiple PDFs**: Combine as many PDFs as you need
- 🔄 **Reorder files**: Easily change the order before merging
- 🌓 **Light/Dark theme**: Choose your preferred theme
- 📱 **Responsive design**: Works on desktop and mobile
- 🆓 **100% free**: No limits, no watermarks, no registration
- 💝 **Support the project**: Help keep this tool free

## 🚀 How to Use

1. Visit [https://universussoft.github.io/pdf-joiner](https://universussoft.github.io/pdf-joiner)
2. Select PDF files (drag & drop or click to browse)
3. Reorder files if needed using the arrows
4. Click "Merge PDFs"
5. Download your merged PDF

## 🔧 Technical Details

This tool uses:
- [PDF-Lib](https://pdf-lib.js.org/) for PDF manipulation
- HTML5 File API for file handling
- Vanilla JavaScript for the UI

## ⚠️ Limitations

- Maximum file size depends on your browser's memory
- Processing very large files may be slow
- Password-protected PDFs cannot be processed
- Some complex PDF features may not be preserved
